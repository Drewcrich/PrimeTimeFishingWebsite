# PrimeTimeFishingWebsite

This repository is deployed to Netlify at `https://primetimefishing.netlify.app/`.

## Files

- `index.html` — site home (includes Transparency section linking to species pages)
- `yellowfin.html` — Yellowfin transparency page
- `bigeye.html` — Bigeye transparency page
- `mahi.html` — Mahi-mahi transparency page
- `blackmarlin.html` — Black Marlin transparency page
- `walleye.html` — Walleye transparency page
- `muskie.html` — Muskie transparency page
- `pike.html` — Northern Pike transparency page
